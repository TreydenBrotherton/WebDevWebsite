Group 1 Questions

What are the principles of analysis and design?
There are twelve basic principles of design: contrast, balance, emphasis, proportion, hierarchy, repetition, rhythm, pattern, white space, movement, variety, and unity.
There are four basic principles of analysis: Measure, Integration, Functional Analysis, and Applications.
What are website architectural requirements?
Here are 8 requirements that create a good website architectural base;
Functional: Core features and functionalities, user interactions, content management, specific functions for the website
Performance: Response times, page load times, server responsiveness
Security: Authentication, authorization, data encryption, protection against common vulnerablities
Scalability: Handle increase traffic and growing user bases
User Experience: desirable user interface, quality of life, user-friendly, responsive, accessible on various devices
Data Managment: How is data stored, retreived and managed, database design, data integirty, data backup
Integration: API's or systems for features, data exchange formats and protocols
Deployment & Maintenance: version control practices, ongoing maintenance and support procedures
What are website design structures, including hierarchy and navigation design?
Design structures include layout, typography and colour schemes for a visually appealing and organized website
navigation design involves menus, buttons and search bars
Prioritize user experience, hierachy and consistency in design to create a user-friendly website
What are user-interface design requirements and production processes?
Some design requirements could include:
Target audience research, Wireframes and prototypes, information architecture, visual design guidelines and responsive design
Some Design Production Procceses could include:
Conceptualization, sketching and wireframing, prototyping, visual design, design hand off, development and testing and quality assurance
